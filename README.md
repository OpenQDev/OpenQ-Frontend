# OpenQ-Frontend 
 
## Boot Locally

If the work you are engaged in doesn't require the Fullstack, you can boot just the OpenQ-Frontend by following these instructions:

### 1 Boot a Local Json Server

```bash
yarn json-server
```

### 2 Boot the App

```bash
yarn boot:local
```

### 3 Access OpenQ on http://localhost:3000
